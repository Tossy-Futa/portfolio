App

【ファイルランダム選択くん（仮）】

  1. フォルダ参照ボタンを押下して、フォルダを選択
  2. ファイル種類（拡張子）を選択
  3. ファイルを選ぶボタンを押す
  4. ファイルを開くボタンを押すと、ファイルがランダムに開かれる

Python(tkinter)
あるフォルダに存在する画像ファイルや動画ファイル等が、何でもいいから表示されてほしい時がよくあります。
そんな時にこれを使うと、適当にランダムでファイルを選択してくれ、表示してくれます。

コード（Git Hub)

【パスワードをクリップボードに入れるくん（仮）】

  1. PlatFormボタンを押す
  2. ログインしたいプラットフォームを選択（タブで選ぶ）
  3. PWPasteボタンを押す
  4. クリップボードにパスワードが入る

Python(Kivy)
通販サイト、証券サイト、エンタメ系サイト等々…で、ログインをする際に必要になるパスワード。
一回一回入力するのは面倒くさい。それに噂によると、パスワード入力時にパスワードを盗み取られる可能性もあるとか。
そんな時にこれを使うと、貼り付けを行う（Ctrl + V)だけでパスワードが入力されます。
